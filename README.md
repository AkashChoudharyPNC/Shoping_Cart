# Pnc Shop

Welcome to Pnc Shop, an online store where you can find a variety of products including smartphones, laptops, headphones, smartwatches, tablets, and more.

## Project Structure

- `index.html`: The main HTML file that contains the structure of the webpage.
- `styles.css`: The CSS file that contains the styles for the webpage.
- `scripts.js`: The JavaScript file that contains the functionality for the shopping cart and product display.
- `products.json`: The JSON file that contains the product data.

## Features

- **Responsive Design**: The website is designed to be responsive and works well on different screen sizes.
- **Dynamic Product Display**: Products are loaded dynamically from a JSON file.
- **Shopping Cart**: Users can add products to the cart, increase or decrease the quantity, and view the total price.
- **Local Storage**: The cart data is saved in local storage, so it persists even after the page is refreshed.

## How to Run

1. Clone the repository to your local machine.
2. Open `index.html` in your web browser to view the website.
